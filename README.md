# Prova Fácil MVP

Protótipo de aplicação para organizar questões enviadas pelos professores e montar provas padronizadas rapidamente.

## O que já funciona

- Escolha de templates da prova
- Cadastro dos dados básicos da avaliação
- Colagem de texto bruto da questão
- Parser simples para enunciado e alternativas
- Montagem da prova com múltiplas questões
- Pré-visualização ao vivo
- Impressão / exportação via navegador
- Persistência local no `localStorage`

## Como abrir

Abra o arquivo `index.html` no navegador ou use uma extensão de servidor local no VS Code.
